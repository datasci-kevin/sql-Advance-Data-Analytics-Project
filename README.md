📊 End-to-End Exploratory Data Analysis (EDA) – Sales Dataset
🚀 Project Summary

This project showcases a structured, end-to-end Exploratory Data Analysis (EDA) built entirely in SQL, following a real-world analytical workflow.

Instead of running random queries, this project demonstrates how to:

Break down a dataset systematically
Extract meaningful business insights
Build reusable and scalable SQL logic

📌 The analysis is organized into modular scripts, simulating how data professionals approach real production datasets.

💼 Business Context

The objective is to analyze a sales dataset to answer key business questions such as:

How are sales evolving over time?
Who are the top customers and products?
Which segments are underperforming?
What trends can inform better decision-making?
🧠 Skills Demonstrated
Advanced SQL (CTEs, Window Functions, Aggregations)
Analytical thinking & problem breakdown
Time series analysis (YoY, trends, cumulative metrics)
Data storytelling through structured queries
Clean, production-style SQL organization
🗂️ Project Structure (Step-by-Step EDA)

Each script represents a stage of the analytical workflow:

01_Database_Exploration.sql       -- Understanding raw data
02_Dimension_Exploration.sql     -- Exploring categorical dimensions
03_Date_Range_Exploration.sql    -- Time boundaries and coverage
04_Measure_Exploration.sql       -- Core metrics (Sales, Quantity, etc.)
05_Magnitude_Analysis.sql        -- Distribution & scale analysis
06_Ranking_Analysis.sql          -- Top/Bottom performers
07_Change_Over_Time.sql          -- Trends & growth patterns
08_Cumulative_Analysis.sql       -- Running totals
09_Performance_Analysis.sql      -- YoY performance evaluation
10_Part_to_Whole_Analysis.sql    -- Contribution analysis
11_Data_Segmentation.sql         -- Grouping & segmentation
12_Report_Customers.sql          -- Customer insights
13_Report_Products.sql           -- Product insights
🔍 Key Insights & Techniques
📅 Time-Based Analysis
Year-over-Year (YoY) comparisons
Trend identification

Tracking business evolution across time
